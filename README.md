# bill-image-data-extraction
This project extracts tabular data from bill images using Groq Vision API

## Technolgies
-Python
-Groq Vision API
-Google Colab

## Workflow
1. Encode image to base64
2. Send image to Groq vision model
3. Extract structured table data

## Sample output
Item price
----------
lorem 1.21
Ipsum 3.12
Dolar 2.31

## Run
1. Add Groq API Key
2. Upload Image
3. Run notebook
